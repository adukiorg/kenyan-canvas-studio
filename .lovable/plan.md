
# DARKCHESA — Digital Painter Portfolio (Web)

A high-end, client-selling interactive portfolio website for a Kenyan digital painter brand. Single-page scroll with smooth section navigation, designed to convert visitors into paying clients.

## Brand & Aesthetic
- **Name:** DARKCHESA
- **Tagline:** "Digital Painter | Sketch • Color • Print • Frame"
- **Palette:** Deep charcoal/near-black background, warm off-white text, **crimson** accent (#9B1B30 / muted blood-red) for highlights, CTAs, dividers
- **Typography:** Elegant serif display (Playfair Display) for headings + clean sans-serif (Inter) for body
- **Feel:** Fine-art gallery — generous whitespace, subtle grain/texture, slow fade-in animations on scroll

## Sections (single-page, anchor nav)

1. **Cover / Hero**
   - Full-viewport dark hero, large serif "DARKCHESA"
   - Tagline beneath, crimson hairline divider
   - Centerpiece: large styled placeholder framed like gallery artwork
   - Sticky top nav (About · Services · Gallery · Process · Pricing · Contact)

2. **About**
   - Two-column: portrait placeholder + bio
   - Short professional bio emphasizing Kenyan artistic heritage, craftsmanship, contemporary digital technique
   - Three small stat callouts (Years creating · Pieces delivered · Happy clients)

3. **Services**
   - 4 elegant cards: Sketching · Digital Coloring · Printing · Framing
   - Each: crimson icon, title, short description, "from KES X" hint

4. **Portfolio Gallery**
   - Filter tabs: All · Sketches · Colored Works · Final Prints
   - Masonry/grid of 9–12 styled placeholder tiles with hover zoom + crimson overlay
   - Each tile labeled (e.g., "Sketch 01 — Portrait Study")

5. **Process**
   - Horizontal numbered timeline (1–6): Concept → Sketch → Color → Print → Frame → Delivery
   - Crimson connecting line, icon per step

6. **Pricing**
   - Clean card/table layout in KES:
     - Print only (A4 / A3 / A2)
     - Standard framed (A4 / A3 / A2)
     - Premium framed (A4 / A3 / A2)
   - "Most popular" badge on one tier in crimson
   - CTA button: "Request a Commission"

7. **Contact**
   - Phone / WhatsApp (click-to-chat), Email, Instagram/TikTok/X handles, Location: Nairobi, Kenya
   - Simple inquiry form (Name, Email, Service interest, Message) — submits to a toast confirmation
   - Footer with brand mark and copyright

## UX Details
- Smooth scroll between sections, sticky nav highlights active section
- Scroll-triggered fade/slide animations (subtle, gallery-paced)
- Fully responsive (mobile-first, hamburger nav under md breakpoint)
- WhatsApp floating button (crimson) always visible for quick client contact
- All artwork uses elegant labeled placeholders ready to swap later

## Out of scope (for now)
- Real backend form submission (toast only — can add Lovable Cloud + email later)
- CMS for editing artworks (artworks defined in code; easy to swap)
- E-commerce checkout
